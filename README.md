# Formação HTML e CSS - Alura

Repositório contendo três projetos práticos desenvolvidos durante o curso de formação em **HTML e CSS** da Alura. Cada projeto foi construído seguindo especificações de design no Figma e aplicando conceitos progressivos de HTML, CSS, Flexbox, Grid e design responsivo.

---

## 📋 Projetos

### **Projeto 1 - Portfolio**

Um portfólio pessoal responsivo desenvolvido com HTML e CSS puro, utilizando Flexbox para layout.

**Características:**
- Design responsivo (Desktop/Mobile)
- Utilização de Flexbox para alinhamento e distribuição de elementos
- Seguindo especificação: [Portfolio Figma](https://www.figma.com/file/NrzJacC887svMVfF9oC2jM/Portfolio-Projeto-2)

**Tecnologias:**
- HTML
- CSS
- Flexbox

---

### **Projeto 2 - Alura Plus (Landing Page)**

Uma landing page para a plataforma Alura Plus, demonstrando o uso de Grid e Flexbox para layouts mais complexos.

**Características:**
- Layout moderno com Grid e Flexbox
- Design responsivo
- Múltiplas seções de conteúdo
- Seguindo especificação: [Alura Plus Figma](https://www.figma.com/file/tFDVyNuKhrT2G03k2dCstW/Alura-Plus---Layout)

**Tecnologias:**
- HTML
- CSS
- Flexbox
- CSS Grid

---

### **Projeto 3 - Mobile First (Biblioteca)**

Uma plataforma de biblioteca digital desenvolvida com abordagem **Mobile First**, incluindo funcionalidades avançadas de interatividade.

**Características principais (conforme commits):**
- Abordagem Mobile First
- Header responsivo com menu hamburger customizado
- Importação de fontes personalizadas
- Barra de pesquisa
- Carrossel de conteúdo com SwiperJS
- Cards de produtos
- Seção de tópicos visitados recentemente
- Sessão de contato e rodapé
- Media queries para tablets e desktop
- Pseudo-classes CSS
- Ajustes responsivos para diferentes tamanhos de tela

**Tecnologias:**
- HTML
- CSS
- JavaScript (SwiperJS)
- Design responsivo com Media Queries
- Mobile First approach

---

## 🎯 Conceitos Aplicados

Ao longo dos três projetos, foram aplicados os seguintes conceitos:

- ✅ Estruturação semântica com HTML
- ✅ Estilização com CSS puro

- ✅ CSS Grid para layouts avançados
- ✅ Design responsivo
- ✅ Abordagem Mobile First
- ✅ Media queries para diferentes breakpoints
- ✅ Pseudo-classes e pseudo-elementos CSS

- ✅ Interatividade básica
